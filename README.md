## Highlights
- **Extreme ocean-tide background** — a Three.js Gerstner-wave sea with towering swells and white foam crests over near-black water, rolling toward the camera and reacting to mouse + scroll.
- **Monochrome UI** — pure black/grey/white theme, silver sheen accents, grayscale photography.
- Animated preloader, custom cursor, magnetic buttons, scroll reveals, animated counters.
- Masonry gallery (true aspect ratio, no cropping) with a full keyboard-navigable lightbox.
- Fully responsive + reduced-motion support.

## Stack
Plain HTML / CSS / JavaScript. Three.js (r128) via CDN. No build step.
Auto-deploys to GitHub Pages (custom domain `ahnaf.ami.bd`) on every push to `main`.

## Run
Open `index.html` in a browser (needs internet on first load for Three.js + fonts).

## Structure
```
index.html      # markup & content
styles.css      # monochrome theme, layout, UI
app.js          # Three.js ocean tide + interactions
assets/1–7.jpg  # gallery frames
```
