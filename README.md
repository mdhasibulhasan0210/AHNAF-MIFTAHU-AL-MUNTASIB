# Ahnaf Miftahu Al Muntasib — Portfolio

A cinematic, color-rich personal portfolio site.

## Highlights
- **Interactive Three.js background** — a rippling particle-wave field tinted across a warm-to-cool palette (amber · coral · violet · teal) that reacts to mouse and scroll.
- **Vivid gradient UI** — deep near-black base `#0a0810`, off-white ink, and a signature amber→coral→violet→teal gradient running through headings, links, stats and accents.
- Drifting color aurora behind the particles, animated preloader, custom cursor, magnetic buttons, scroll reveals, animated counters.
- Masonry gallery with full lightbox (keyboard nav) using 7 frames, images shown in full color with a hover saturation pop.
- Fully responsive + reduced-motion support.

## Stack
Plain HTML / CSS / JavaScript. Three.js (r128) via CDN. No build step.

## Run
Open `index.html` in a browser (needs internet on first load for Three.js + fonts).

## Structure
```
index.html      # markup & content
styles.css      # theme, layout, UI (color gradient system)
app.js          # Three.js animation + interactions
assets/1–7.jpg  # gallery frames
```
